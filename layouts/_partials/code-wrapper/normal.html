{{- /* Normal Code Block Wrapper */ -}} {{- $codeblock := .Opts -}} {{- $wrapper := .Wrapper -}} {{- $editBtn := "" -}} {{- $copyBtn := "" -}} {{- $foldIcon := dict "Class" "arrow fa-solid fa-chevron-right fa-fw" | partial "plugin/icon.html" -}} {{- $titleEl := printf `%s` $foldIcon -}} {{- $ellipsisBtn := printf `%s` (dict "Class" "fa-solid fa-ellipsis-h fa-fw" | partial "plugin/icon.html") -}} {{- /* Code Header */ -}} {{- if $codeblock.title -}} {{- $titleEl = printf `%s%s` $foldIcon $codeblock.title -}} {{- end -}} {{- if $codeblock.editable -}} {{- $editIcon := dict "Class" "fa-solid fa-pen-to-square fa-fw" | partial "plugin/icon.html" -}} {{- $editBtn = printf `%s` (T "assets.editUnLockTitle") $editIcon -}} {{- end -}} {{- if $codeblock.copyable -}} {{- $copyIcon := dict "Class" "fa-regular fa-copy fa-fw" | partial "plugin/icon.html" -}} {{- $copyBtn = printf `%s` (T "assets.copyToClipboard") $copyIcon -}} {{- end -}} {{- $codeHeader := printf `
%s%s%s%s
` (lower .Type) $titleEl $ellipsisBtn $editBtn $copyBtn -}} {{- $wrapper = replaceRE `()` (printf "%s$1" $codeHeader) $wrapper -}} {{- return $wrapper -}}
